grid example

CSS Grid is a powerful two-dimensional layout system for the web. It allows you to create complex layouts using rows and columns with minimal code. Unlike Flexbox, which is designed for one-dimensional layouts (either in a row or a column), Grid can handle both dimensions simultaneously, making it ideal for creating grid-based layouts.
