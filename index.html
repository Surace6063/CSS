Diffrent Ways to color

This paragraph is styled using a HEX color value (#ff5733). HEX (#rrggbb): Specifies color using a hexadecimal code. Example: #ff5733 (Red: ff, Green: 57, Blue: 33).

This paragraph is styled using an RGB color value (rgb(0, 123, 255)).RGB (rgb(r, g, b)): Defines color using red, green, and blue components (0–255). Example: rgb(0, 123, 255) (0 Red, 123 Green, 255 Blue).

This paragraph is styled using an HSL color value (hsl(120, 70%, 40%)).HSL (hsl(h, s, l)): Defines color using hue (h in degrees), saturation (s in %), and lightness (l in %). Example: hsl(120, 70%, 40%) (120° is green, 70% saturation, 40% lightness).

css box model

  1. Content: The innermost part of the box where the text or elements reside.
  2. Padding: The space between the content and the border.
  3. Border: The visible edge around the padding.
  4. Margin: The outermost space that separates the element from others.
